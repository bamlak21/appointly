# Appointly

A modern appointment scheduling application built with Next.js, featuring AI-powered assistance and real-time calendar management.

## Features

- Interactive calendar for scheduling appointments
- AI-powered appointment suggestions and management
- Real-time updates with Supabase
- Responsive design with Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 15
- TypeScript
- Supabase
- Tailwind CSS
- AI integration
